# 運用ガイド

## ファイル構成

| ファイル | 役割 |
|---------|------|
| `header.php` | 全ページ共通のヘッダー（グローバルナビ） |
| `footer.php` | 全ページ共通のフッター＋モバイルナビ |
| `.htaccess` | .html でも PHP を動かすための設定 |
| `各ページ/index.html` | 各ページ固有のコンテンツ |

## ナビゲーションにページを追加する場合

1. `header.php` を開いて、`<li>` タグでリンクを追加
2. `footer.php` にも同じリンクを追加（フッターナビ＋モバイルナビの2箇所）
3. **この2ファイルだけサーバーにアップロードすれば全ページに反映されます**

## 新規ページを追加する場合

1. 新しいフォルダ（例: `newpage/`）を作成
2. `index.html` を作成（既存ページをコピーして編集するのが楽）
3. ヘッダー部分は `<?php include "../header.php"; ?>` と書くだけでOK
4. フッター部分は `<?php include "../footer.php"; ?>` と書くだけでOK
5. ナビに追加する場合は上記「ナビゲーションにページを追加する場合」も実施
6. **新規ページの `index.html` + `header.php` + `footer.php` の3ファイルだけアップ**

## 注意事項

- `header.php` と `footer.php` 内のパスはすべてルート相対（`/assets/xxx`）で記述してください
- 各ページの `index.html` 内のパスは従来通り相対パス（`../assets/xxx`）のままでOKです
- ローカルで確認する場合は `php -S localhost:8000` でPHPサーバーを起動してください
